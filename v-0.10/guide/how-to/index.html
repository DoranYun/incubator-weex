Updated time: 30/03/2017

How to preview weex code in browser ?

weex-toolkit

We strongly suggest you use weex-toolkit to preview weex code in your browser. This tool is Node.JS based, so you need to install Node at first. Please download and install latest stable version of Node from https://nodejs.org/en/download/stable/. Then you can install weex-toolkit using npm install:

Check that the toolkit does work by typing weex in the command line. Normally you should see the following help text:
